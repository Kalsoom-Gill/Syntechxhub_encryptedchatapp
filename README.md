# Encrypted Chat App

## Description

This is a simple encrypted chat application developed in Python. It uses TCP socket programming for communication and the Cryptography library (Fernet) to encrypt and decrypt messages securely.

## Features

* Client-Server Communication
* Encrypted Messages
* Multiple Client Support
* Chat Logging

## Technologies Used

* Python
* Socket Programming
* Cryptography (Fernet)

## How to Run

1. Install the required library:

```
pip install cryptography
```

2. Generate the encryption key:

```
python generate_key.py
```

3. Start the server:

```
python server.py
```

4. Open another terminal and start the client:

```
python client.py
```
